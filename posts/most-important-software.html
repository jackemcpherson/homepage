25 Most Important Post-Punch-Card Software

  1. UNIX The operating system that taught every other operating system how to behave 1969
  2. C The language everything else is written in, whether it knows it or not 1972
  3. SQL Made data queryable by anyone who could think in tables 1974
  4. VisiCalc The first killer app — gave ordinary people a reason to buy a computer 1979
  5. Email / SMTP Async communication at scale, still unmatched for reach 1982
  6. TCP/IP stack The protocol suite that became the internet's nervous system 1983
  7. GNU toolchain Made free software a practical reality, not just an ideology 1983
  8. Macintosh System Software Proved the GUI wasn't a research curiosity 1984
  9. Photoshop Turned the computer into a creative tool for an entire industry 1990
  10. Linux kernel Free UNIX for everyone — now runs most of the world's servers 1991
  11. Mosaic / Netscape Made the web visual and gave civilians a reason to go online 1993/1994
  12. Windows 95 Put a PC on every desk and a Start menu on every screen 1995
  13. Apache HTTP Server The web server that powered the first wave of the web 1995
  14. Java Write once, run anywhere — enterprise software's lingua franca 1995
  15. JavaScript Shipped in 10 days, now runs in every browser on Earth 1995
  16. PageRank / Google Search Organized the internet and made finding things trivial 1998
  17. Wikipedia Proved strangers could build an encyclopedia together 2001
  18. BitTorrent Decentralized file distribution at planetary scale 2001
  19. Google Maps Killed the paper map and invented the modern web app along the way 2005
  20. Git Distributed version control that changed how all software gets built 2005
  21. YouTube Broadcast yourself — democratized video for a billion creators 2005
  22. AWS Rented infrastructure that let startups skip the server room 2006
  23. iOS / Android Put a supercomputer in every pocket on the planet 2007/2008
  24. WhatsApp Replaced SMS for two billion people overnight 2009
  25. GPT / ChatGPT Made machines that can read, write, and reason — sort of 2018/2022
